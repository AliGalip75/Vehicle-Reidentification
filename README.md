
# Vehicle Reidentification

Bu projenin amacı, bir kamerada gözüken aracın başka bir kamerada tanınması problemi (Reidentification) için eğitim kodu olarak kullanılmasıdır.

# Vehicle Re-Identification (Güncellenmiş 2025)


VeRi-776 veriseti ile derin öğrenme tabanlı model geliştirme.

## Yenilikler (2025)
- Python 3.9+ ve PyTorch 2.x uyumluluğu
- Eski ve kullanılmaz hale gelen(deprecated) kısımlar yenilendi.
- Resnet50-ibn ve Swin Transformer modelleri yeni kütüphane sürümlerine uygun hale getirildi.
